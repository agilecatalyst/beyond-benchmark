### How work is done here

Much drafting, critique, and scaffolding is done **together with** large models (meccano / alternative intelligence-as-instrument). That is process fact, not a mind claim.

| We credit | We do not claim |
| --- | --- |
| Human operator judgment, logging, and final responsibility | That the model “discovers,” “likes,” or has phenomenal experience |
| Model output as heavy lifting on text, structure, and critique shapes | That useful collaboration proves consciousness or agency-as-person |
| Reproducible runs, word counts, UI times | Stories that put the monkey in the tree because it likes bananas |

If it is not in a log or a control file, it is not evidence.
