# 03 — Framing: thinking machine → intelligence amplification

## Preferred stack

1. **Primary:** *thinking machine (without intent)* → **intelligence amplification**  
   - Fits deliberation proxies, J-space / functional emotion *machinery*  
   - Keeps goals with the human; no agency smuggling  
2. **Secondary (oddities):** *alien organization* / *non-human workspace* — not default *alien mind*  
3. **Avoid as main label:** mind, soul, conscious partner

## Why not “alien mind” as default?

Strong for “don’t expect a human psyche.” Weak as workframe for IA (amplifies *human* intent). Pulls toward personhood / hype.

## One-liner for this repo

> We treat the system as a **thinking machine without intent** — an intelligence amplifier whose internal organization can be oddly non-human, without granting it a mind.
